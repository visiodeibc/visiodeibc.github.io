Twitter-clone-1 (fastapi setup)

이 글은 Twitter Clone 사이드 프로젝트 가즈아아아아아의 일부로서 풀스텍을 배워보고 싶은
노력을 담은 글이다. Developing a Single Page App with FastAPI and Vue.js를 참고해서 작성하였으며
이 글들을 따라하면 vue-javascript (front), fastapi-python (back), docker-compose(deployment)들의 대략적인 부분들을 배울 수 있을것이다.

#목차

  • 1. 폴더 구조
  • 2. fastapi 코드 작성
  • 3. back end Dockerfile
  • 4. docker-compose.yml 작성
  • 5. docker-compose build 해보기
  • 6.CORSMiddleware 미들웨어 사용

#1. 폴더 구조

이번 프로젝트에서는 기본적으로 docker-compose와 docker를 사용하여 deployment를 관리한다.
초반 세팅을 위해서 필요한 fastapi 환경 설정을 할 수 있는 도커 파일을 만들어야 한다.

각 폴더에 Dockerfile이 정의되고 해당 Dockerfile은 docker-compose.yml에서 조합되어 사용될 것이다.

#5. docker-compose build 해보기

docker-compose up -d --build 명령어를 docker-compose.yml 디렉토리에 가서 실행하면 막 docker가 말리고
실행되기 시작할 것이다. 실행해서
http://127.0.0.1:5000/ 에서는 Hello World!를 확인할 수 있고
http://localhost:5000/docs 에서는 Swagger document를 확인할 수 있다.

정의할 수 있도록 해준다. 이를 통해 지정되지 않은 서비스에서 들어오는 요청을 차단함으로서 보안을 강화할 수 있다.
